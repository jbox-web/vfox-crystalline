# vfox-crystalline

[![CI](https://github.com/jbox-web/vfox-crystalline/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-crystalline/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [crystalline](https://github.com/elbywan/crystalline)

## Install

```
mise plugin install crystalline https://github.com/jbox-web/vfox-crystalline.git
```
